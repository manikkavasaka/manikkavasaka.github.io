## 1. Visual Design System

### Color Palette (Brand Colors)
- **Primary:** #ff8c00 (Orange) - CTAs, highlights
- **Secondary:** #d4af37 (Gold) - Accents
- **Background:** #0a0b10 (Deep Black)
- **Light:** #ffffff (White) - Text, cards

### Image Style Guidelines

#### Photography Style
- **Tone:** Professional, modern, aspirational
- **Color Grading:** Warm, slightly desaturated to match brand colors
- **Composition:** Rule of thirds, depth focus
- **People:** Diverse, authentic, in work/business contexts
- **Quality:** Minimum 1200px wide, 72+ DPI

#### Illustration Style
- **Type:** Minimalist, line-based
- **Color:** 2-3 colors max (primary + secondary + one accent)
- **Stroke Weight:** 1-2px consistent
- **Style:** Modern, flat design with subtle depth
- **Usage:** Concepts, features, abstract ideas

#### Icon Style
- **Type:** Outlined, 24-48px
- **Stroke:** 1.5-2px
- **Color:** Primary orange or white
- **Consistency:** Rounded corners, uniform sizing

### Image Quality Standards
- **Resolution:** Minimum 1200x800px (landscape) or 800x1200px (portrait)
- **Format:** WebP (primary), JPG fallback, PNG for transparency
- **Compression:** Optimized for web (<200KB per image)
- **Aspect Ratios:** 
  - Full-width hero: 16:9
  - Service cards: 4:3
  - Industry icons: 1:1
  - Team photos: 1:1 (square)
  - Feature images: 3:2

## 6. CSS Image Styling

### Responsive Images
```css
img {
    max-width: 100%;
    height: auto;
    display: block;
}

/* Hero Images */
.hero-image {
    width: 100%;
    height: 600px;
    object-fit: cover;
    object-position: center;
}

@media (max-width: 768px) {
    .hero-image {
        height: 300px;
    }
}

/* Service Cards */
.service-card img {
    width: 100%;
    aspect-ratio: 4 / 3;
    object-fit: cover;
    border-radius: 12px;
    transition: transform 0.3s ease;
}

.service-card:hover img {
    transform: scale(1.05);
}

/* Team Photos */
.team-photo {
    width: 250px;
    height: 250px;
    border-radius: 50%;
    object-fit: cover;
    border: 3px solid var(--primary);
}

/* Icons */
.service-icon {
    width: 60px;
    height: 60px;
    object-fit: contain;
}

/* Image with Overlay */
.image-overlay {
    position: relative;
    overflow: hidden;
    border-radius: 12px;
}

.image-overlay::before {
    content: '';
    position: absolute;
    inset: 0;
    background: linear-gradient(135deg, rgba(255, 140, 0, 0.2), rgba(212, 175, 55, 0.1));
    z-index: 1;
}
```

## 7. Image Loading Strategy

### Lazy Loading
```html
<img src="service-image.webp" 
     alt="Service description"
     loading="lazy"
     width="400" 
     height="300">
```

### Picture Element for Responsive Images
```html
<picture>
    <source srcset="image-large.webp" media="(min-width: 1024px)">
    <source srcset="image-medium.webp" media="(min-width: 768px)">
    <img src="image-small.webp" alt="Description">
</picture>
```

### WebP with JPG Fallback
```html
<picture>
    <source srcset="image.webp" type="image/webp">
    <img src="image.jpg" alt="Description">
</picture>
```
