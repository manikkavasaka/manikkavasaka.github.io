🎓 QUICK CUSTOMIZATION GUIDE:
═══════════════════════════════════════════════════════

CHANGE COLORS:
  File: public/style.css (Line 8-20)
  Find: --primary: #ff8c00;
  Change to: --primary: #YOUR_COLOR;

UPDATE LOGO:
  File: Any HTML file
  Find: <a href="/" class="logo">MK<span>SHOPZONE</span></a>
  Change to: Your logo text

ADD GOOGLE ANALYTICS:
  File: Any HTML file (before </body>)
  Add: <script async src="..."></script>

CHANGE FONTS:
  File: public/style.css (Line 24)
  Find: --font-main: 'Inter'
  Change to: your font name

ADD NEW PAGE:
  1. Copy any service page HTML
  2. Update content
  3. Add link to navigation
  4. Update vite.config.js
