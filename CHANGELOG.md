## 🎨 Customization Guide

### Change Brand Colors
Edit CSS variables in `public/style.css`:
```css
:root {
  --primary: #ff8c00;      /* Change to your brand color */
  --secondary: #d4af37;    /* Change accent */
}
```

### Update Content
- Edit text directly in `index.html`
- Update service descriptions
- Modify testimonials
- Change pricing

### Add New Sections
1. Create new `<section>` in HTML
2. Style with CSS following existing patterns
3. Add JavaScript interactivity if needed
