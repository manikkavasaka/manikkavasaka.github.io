# MK ShopZone - Digital Marketing Agency

🚀 **Live Deployment:** [https://mkshopzone.me/](https://mkshopzone.me/)

## Features
- Premium UI with glassmorphism and animations
- Lead capture forms with WhatsApp integration
- Automated Tamil registration messages
- Admin dashboard for lead management
- Full-service agency sections (SEO, Ads, Web Dev)

## Tech Stack
- React + Vite
- Tailwind CSS
- Node.js Express Backend
- Twilio WhatsApp API
- GitHub Actions for Deployment
